# [TOP-10] Vulnerable-Web-Application [ENG/RUS]

## [RUS] Создание/взлом бэкенда на Java

В данной теме будет рассмотрены 2 персонажа: Бэкенд Разработчик и Хакер

> [!NOTE]
> ### От лица Бэкенд Разработчика будет представлено:
>
> - Демонстрация продуктов с бэкендом на Java с 1 уязвимостью из OWASP Top 10 (возможно и другие)
> - Возможные усложнения и проверка валидации после успешного взлома Хакера

> [!CAUTION]
> ### От лица Хакера будет представлено:
>
> - Отчет с подробным обьяснением типа уязвимости и самого процесса взлома сайта
> - Советы по улучшению и показ БЕЗОПАСНОГО блока кода, за место уязвимого участка

-----

## Задачи и Цели:

### Цель:

Создание контролируемой среды для демонстрации критических уязвимостей из списка OWASP Top 10 и методов их устранения в среде Java Spring

### Задачи:

 - Показать, как использование простых переборов может обернутся в критическую уязвимость
 - Продемонстрировать, какие типы уязвимостей имеются
 - Рассказать, какие методы по устранению уязвимостей существуют
 - Научить, как эксплуатировать эти уязвимости и как от них защищаться

-----
